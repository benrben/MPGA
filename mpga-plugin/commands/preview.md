# /mpga:preview

Serve the current milestone prototype locally. Localhost only. Safe. Fast. Tremendous.

## Steps

1. Run `mpga preview`
2. Bind to `127.0.0.1:8420` by default
3. Serve prototype files for the current milestone (`mpga milestone show` to find it)
4. Optionally pass `--open` to open the browser automatically

## Usage
```
/mpga:preview
/mpga:preview --port 8421 --open
```
