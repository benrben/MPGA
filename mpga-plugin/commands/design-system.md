# /mpga:design-system

Build or audit the design token layer. No hardcoded visual chaos.

## Steps

1. Initialize tokens:
   `mpga design-system init`
2. Add a token:
   `mpga design-system add-token --name primary --value "#2563eb" --category color`
3. Inspect the component catalog:
   `mpga design-system catalog`
4. Audit token compliance:
   `mpga design-system audit`

## Usage
```
/mpga:design-system init
/mpga:design-system audit
```
