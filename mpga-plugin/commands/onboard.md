# /mpga:onboard

Guided codebase tour from INDEX.md outward for new developers or new sessions. Very, very special tour, believe me. Evidence First.

## Steps

1. Run `mpga status` — present project identity
2. Explain project type, size, and key languages
3. Walk through the scope registry
4. Highlight the dependency graph from GRAPH.md
5. Show active milestone and current board state
6. Suggest starting points and ask where the user wants to explore — Tremendous codebase, enjoy!

## Usage
```
/mpga:onboard
```
