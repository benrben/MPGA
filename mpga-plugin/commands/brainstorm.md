# /mpga:brainstorm

Socratic design refinement before writing any code.

## Steps

1. DO NOT jump into code — ask clarifying questions first
2. Use the Socratic method to refine the design:
   - What problem does this solve?
   - Who are the users and constraints?
   - What does success look like?
   - What existing patterns should we follow?
   - What are the edge cases?
3. Present design in sections, get approval on each
4. Save approved design to `MPGA/milestones/<id>/DESIGN.md`

## Usage
```
/mpga:brainstorm I want to add real-time notifications
```
